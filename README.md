# 🛒 E-Commerce Customer & Sales Analytics (Portfolio Project)

## 📌 Project Overview
This project showcases an end-to-end data pipeline for an E-Commerce business. It covers data extraction from SQL, cleaning messy real-world datasets with Python, and delivering business insights via an interactive Excel dashboard.

## 🛠️ Tech Stack
* **SQL:** Advanced Left Joins for data integration.
* **Python (Pandas):** Handling missing values (NaN), removing duplicates, and fixing mixed date formats.
* **Excel:** Professional Executive Dashboard with Pivot Tables and Slicers.

## 🚀 Key Achievements
* Integrated multiple relational tables from a SQLite database.
* Solved complex data quality issues like "Mixed Date Formats" and "Null Values".
* Created a dynamic visualization allowing for city-wise and product-segment analysis.
